# ms2fft0
Get a signal, FFT it, and output the amplitude and phase.

# Clone
```bash
$ git clone --recursive https://github.com/Cuda-Chen/ms2fft0.git
```

# Compile and Run
```
$ make
$ 
```

# Output Format
```
<fs>, <amplitude>, <phase>
```
NOTE: `<amplitude>` is un-normalized, `<phase>` is in radian.
